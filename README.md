# BaZi Fortune Teller | 八字命理

Interactive BaZi (Four Pillars of Destiny) fortune telling web app. Next.js + TypeScript + Tailwind CSS.

## Features

- **Full BaZi Calculation Engine** — accurate four-pillar computation with solar term boundaries, 60 stem-branch cycle, hidden stems
- **Light / Dark Theme** — toggle between themes; light is default
- **Year-by-Year Timeline** — detailed annual analysis with star ratings, element colors, dayun section dividers, and adjustable year range
- **Day Master Analysis** — personality archetype, strength assessment
- **Five Elements Chart** — Recharts bar visualization
- **Ten Gods Analysis** — complete stem mapping
- **Decade Luck Pillars (大运)** — forward/reverse calculation
- **Life Area Readings** — career, relationships, health, trajectory
- **Current Year Forecast**
- **Compatibility Module (合婚)** — two-person comparison with scoring
- **Bilingual** — English / 中文 toggle

## Getting Started

```bash
npm install
npm run dev
```
